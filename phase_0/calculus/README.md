## Guide

線性代數教你「**空間怎麼被變換**」，微積分教你「**東西怎麼變化**」——這兩件事就是深度學習的雙腳。

* 用 3Blue1Brown《微積分的本質》打底，先建立**幾何直覺**再進公式。導數是「割線變切線的極限」，不是「公式背一背」；積分是「把變化累加起來」，不是「面積公式」。
* 看完影片**立刻打開 Jupyter** 跟著做：用 NumPy 算數值導數、用 matplotlib 畫切線動畫。每一個概念都自己手動算一次、畫一次，公式會「自己長出來」。
* 為什麼這個順序對深度學習特別重要？因為：
    * **反向傳播 = 鏈式法則**
    * **梯度下降 = 多變量導數**
    * **牛頓法 / Adam / L-BFGS = 二階導數**
    * **損失函數逼近 = 泰勒展開**
  
  微積分不是預備知識，是 DL 的引擎。
